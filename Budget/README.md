# Budget Appartement

Site statique pour planifier un budget d'appartement, séparer les dépenses communes et suivre les paiements.

## Développement local

```bash
npm run dev
```

Ouvre ensuite `http://localhost:3000`.

## Build Vercel

```bash
npm run build
```

Le dossier `dist` contient seulement les fichiers publics à déployer.

## Déploiement Vercel

Depuis ce dossier:

```bash
vercel
```

Pour la production:

```bash
vercel --prod
```
